# 🌍 Wanderlust – Airbnb Clone

A full-stack web application for discovering, creating, and reviewing rental listings. Built using **Node.js, Express, MongoDB, and EJS** following **MVC architecture**, with secure authentication, image uploads, and interactive maps.

---

## 🚀 Features

- 🏠 Create, edit, and delete property listings  
- 📝 Add and manage reviews  
- 🔐 User authentication (Register/Login) using Passport.js  
- 🛡️ Role-based authorization (only owners/authors can modify/delete)  
- ☁️ Image upload and storage using Cloudinary  
- 🗺️ Interactive maps using Mapbox (location-based display)  
- ⚠️ Centralized error handling with custom **404 (Not Found)** and **500 (Server Error)** pages  
- ✅ Server-side validation using Joi  
- 💬 Flash messages for success and error feedback
